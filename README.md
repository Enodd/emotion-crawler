# Important notes
Before doing anything in this repo make sure that you have installed NODE.js 18.17 or later       
Once you have it installed all you need is to clone this repo then enter directory and exec

```bash
npm install
#or
pnpm install
```

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
